Bender

Bender is a declarative Web application framework. It is a tool to build Web application front-ends using a markup language for application and component description, a Javascript runtime engine for Web browsers and SVG players, and a library of Javascript support code and predefined components. Bender is free software, released under the Apache License v2.0 and currently under active development.

Without further ado, let's see a very simple Bender application. The complete XML description is reproduced below; to run this application (or any other application mentioned below) in your browser, just click on the ⚐ sign.

Welcome to Bender!

This example shows the main elements of a Bender application:

  • the app element is the container for the description of a complete Bender application;
  • the view element describes how the application is rendered; in this case we use HTML;
  • the watch element is used to react to changes in a component attribute or property, or to events sent by a component: in this case, when we get a @pushed event from the button component, an alert box shows up.

One of the main purposes of Bender is to facilitate the creation of Web applications through reusable components. In the above example, we include a pre-defined button component in our view (notice that the view description mixes HTML elements, which are rendered as is, with the Bender-specific element component, the view of which will then be rendered.) Above is the description of an HTML push button, that can respond to both mouse and touch events (and thus works on both desktop or touch screen environments out of the box.) It reacts to being pushed and released, so that its state can be visualized, but abstracts this from the including application or component by simply sending a @pushed event when the button was clicked or tapped.
