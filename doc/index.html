Bender

Bender is a Web application framework consisting of a markup language for application and component description, a Javascript runtime engine for Web browsers and SVG players, and a library of Javascript support code and predefined components. Bender is free software, released under the Apache License v2.0 and currently under active development.

Bender follows loosely the Model-View-Controller pattern, where models can be seen as data stored on the application back-end (e.g. in a database server-side, or using HTML5 Web storage facilities client-side), views are concrete nodes in a target DOM tree rendered by the host browser or viewer, and controllers are Javascript objects sending and receiving events, and connected to other parts of the application through outlets.

This example shows the main elements of a Bender application:

  • the app element is the container for the description of a complete Bender application;
  • the view element describes how the application is rendered; in this case we use HTML;
  • the controller element describes a controller object that in this case listens to events from a button component.

Running a Bender application consists of loading its description and related resources, such as included components, external scripts and stylesheets; creating prototypes for components and controllers; rendering views in the host browser; then setting up connections between concrete nodes and controller instances.

One of the main purposes of Bender is to facilitate the creation of Web applications through reusable components. In the above example, we include a pre-defined button component in our view (notice that the view description mixes HTML elements, which are rendered as is, with the Bender-specific element component, the view of which will then be rendered.)
