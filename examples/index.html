Bender examples

  • Create a context tree, and that's it.
  • Hello, world! — build a minimal Bender tree that can display a message.
  • Hello, world!/text only — build a minimal Bender component that can display a text-only message.
  • Hello, world!/use — load a minimal Bender component that can display a message.
  • Error loading an unavailable component.
  • Multiple contexts — create multiple Bender contexts, each with their own component, in the same host document.
  • Simple content — simple test of content and default content.
  • Property in attribute — the square attributes (position, size, color) are set using properties.
  • Show property — show the value of a property in a text node; try setting u._instance.properties.foo in the console and see the live updates.
  • Show computed property value — show a computed property in a text node
  • Chain properties — y depends on x, and z depends on both x and y.
